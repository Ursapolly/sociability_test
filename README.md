Небольшой тест на общительность

# Требования

Ruby версии 2.4.3 и выше

# Установка

Скачайте репозиторий

Запустите в терминале с помощью интерпретатора ruby файл main.rb 

В качестве аргумента передайте ваше имя

```ruby main.rb Имя```
